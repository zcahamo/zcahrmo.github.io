Question 1. Calculate

  • a) \(3^{2021}\mod 37\)

  • b) \(3^{2021}\mod 407\)

  

  • a) 37 is prime. By Fermat’s Little Theorem,

    \(\seteqnumber{0}{0.}{0}\)

    \begin{alignat*} {2} 3^{36}&\equiv 1&\mod 37 \end{alignat*} Since \(2021=34\times 56+5\),

    \(\seteqnumber{0}{0.}{0}\)

    \begin{alignat*} {2} 3^{2021}&=(e^{36})^{56}\cdot 3^5\equiv 3^5&\mod 37\\ &\equiv 3^2\cdot 3^3&\mod 37\\ &\equiv 9\cdot (-10)&\mod 37\\ &\equiv 21&\mod 37 \end{alignat*}

  • b) 407 is not prime, \(407=11\times 37\). By the same method as above

    \(\seteqnumber{0}{0.}{0}\)

    \begin{alignat*} {2} 3^{2021}&\equiv 3&\mod 11 \end{alignat*} The answer may be spotted to be

    \(\seteqnumber{0}{0.}{0}\)

    \begin{alignat*} {2} 21+37=58&\equiv 3&\mod 11 \shortintertext {so} 3^{2021}&\equiv 58&\mod 407 \end{alignat*}

    Alternatively, look for a number that satisfies

    \(\seteqnumber{0}{0.}{0}\)

    \begin{align*} x&\equiv 3\mod 11\\ x&\equiv 21\mod 37 \end{align*} so

    \(\seteqnumber{0}{0.}{0}\)

    \begin{align*} x&=21+37k\\ x&=-1+4k\mod 11 \end{align*} So a solution is when \(k=1\) to give, giving \(x=58\)

    Otherwise, use Chinese Remainder Theorem. Euclid’s backwards gives

    \[ -11\times 10+3\times 37=1 \]

    Then

    \[\begin {array}{rllrll} -110&\!\!\!\!\equiv 0&\mod 11&111 &\!\!\!\!\equiv 0&\mod 37\\ -110&\!\!\!\!\equiv 1&\mod 37&\qquad 111&\!\!\!\!\equiv 1&\mod 11 \end {array}\]

    So the solution is

    \[ -110\times 21+3\times 111=-1977\equiv 58\mod 407 \]

Question 2. Solve the congruence

\[ x^7\equiv 4\mod 111 \]

   Use Fermat’s Little Theorem to solve the congruence modulo 3 and 37.

\(\seteqnumber{0}{0.}{0}\)

\begin{alignat*} {2} x^7&\equiv 4&\mod 3\\ x^7&\equiv 1&\mod 3\\ (x^2)^3x\equiv x&\equiv 1&\mod 3\\ \text {and}\quad x^7&\equiv 4&\mod 37 \end{alignat*} This is a little harder. Use

\(\seteqnumber{0}{0.}{0}\)

\begin{alignat*} {2} x^{36}&\equiv 1&\mod 37 \end{alignat*} \(\hcf (7,36)=1\), so find \(c\) such that

\(\seteqnumber{0}{0.}{0}\)

\begin{alignat*} {2} 7c&\equiv 1&\mod 36\\ c&\equiv -5&\mod 36\\ c&\equiv 31&\mod 36 \end{alignat*} Therefore, take the original congruence and raise it to the \(31^\text {st}\) power.

\(\seteqnumber{0}{0.}{0}\)

\begin{alignat*} {2} x^{7\cdot 31}\equiv 1&\equiv 4^{31}&\mod 37 \end{alignat*} Using FLT again gives

\(\seteqnumber{0}{0.}{0}\)

\begin{alignat*} {2} x&\equiv 4^{36}4^{-5}&\mod 37\\ &\equiv (4^5)^{-1}&\mod 37\\ &\equiv (4^3\times 4^2)^{-1}&\mod 37\\ &\equiv (-10\times 4\times 4)^{-1}&\mod 37\\ &\equiv (-3\times 4)^{-1}=(-12)^{-1}&\mod 37\\ x&\equiv 3&\mod 37 \end{alignat*} Alternatively,

\(\seteqnumber{0}{0.}{0}\)

\begin{alignat*} {2} x&\equiv 4^{31}&\mod 37\\ &\equiv 4^{1+2+4+8+16}&\mod 37\\ &\equiv 4\times 16\times 16^2\times 16^4\times 16^8&\mod 37\\ &\equiv 4\times 16\times 34\times 34^2\times 34^4&\mod 37 \end{alignat*} Continuing as such gives

\(\seteqnumber{0}{0.}{0}\)

\begin{alignat*} {2} &\equiv 4\times 16\times 34\times 9\times 7&\mod 37\\ &\equiv 3&\mod 37 \end{alignat*}

Use CRT. Euclid’s backwards gives

\[ 37\times 1+(-3)\times 12=1 \]

Then

\[\begin {array}{rllrll} 37&\!\!\!\!\equiv 0&\mod 37& -36&\!\!\!\!\equiv 1&\mod 37\\ 37&\!\!\!\!\equiv 1&\mod 3&\qquad -36&\!\!\!\!\equiv 0&\mod 3 \end {array}\]

so

\(\seteqnumber{0}{0.}{0}\)

\begin{align*} x&=1\times 37+3\times (-36)\\ &=37-108=-71\\ &\equiv 40\mod 111 \end{align*}

Question 3. How many solutions has the congruence

\[ x^2\equiv 1\mod (3\times 5\times 7\times 11\times 13) \]

  

For odd prime \(p\),

\(\seteqnumber{0}{0.}{0}\)

\begin{alignat*} {2} x^2&\equiv 1&\mod p\\ x^2-1&\equiv 0&\mod p\\ (x+1)(x-1)&\equiv 0&\mod p\\ \implies x&\equiv \pm 1&\mod p \end{alignat*} This is true for each of the primes \(3,5,7,11,13\). By CRT, the congruence is equivalent to the simultaneous equations

\[ \begin {cases} x^2\equiv 1\mod 3\\ x^2\equiv 1\mod 5\\ x^2\equiv 1\mod 7\\ x^2\equiv 1\mod 11\\ x^2\equiv 1\mod 13 \end {cases} \]

So by choosing different solutions to the congruence for each prime \(p\), there are \(2^5=32\) solutions modulo \(3\times 5\times 7\times 11\times 13\).

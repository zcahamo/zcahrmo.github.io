Question 1. Does the series

\[ \sum _{m=0}^\infty m! \]

converge \(p\)-adically for every prime \(p\)?   

So this series converges to some number \(x\) modulo \(p,p^2,\dots \). So

\[ x=d_0+d_1p+d_2p^2+\cdots \]

To see how to solve this, consider the sum

\(\seteqnumber{0}{0.}{0}\)

\begin{align*} \sum _{m=0}^\infty m!&=1+1+2+6+24+\cdots \\ &\equiv 1+1+0+0+0+\cdots \mod 2\\ &\equiv 1+1+2+2+0+0+\cdots \mod 4 \end{align*} This works for any \(n\in \N \), so works for any natural number.

Then he talks about binary and decimal representations for some reason.

For a series to converge \(p\)-adically, require \(\nu _p(a_n)\to \infty \). Clearly this is true for \(a_n=m!\). If \(m\ge p^k\), then \(p^k\mid p^k!\mid m!\), thus \(\nu _p(m!)\ge \nu _p(p^k!)\ge k\) for

\[ k=\floor {\log _p(m)}=\floor {\frac {\log {m}}{\log p}}\ge \frac {\log {m}}{\log p}-1\to \infty \]

Consider the computation of \(\nu _p(m!)\).

\(\seteqnumber{0}{0.}{0}\)

\begin{align*} \nu _p(m!)&=\floor {\frac {m}{p}}+\floor {\frac {m}{p^2}}+\floor {\frac {m}{p^3}}+\cdots \\ &=\sum _{k= 1}^\infty \floor {\frac {m}{p^k}}\\ &\le \sum _{k= 1}^\infty \frac {m}{p^k}= m\sum _{k= 1}^\infty \frac {1}{p^k}\\ \nu _p(m)&\le \frac {m}{p-1}\to \infty \end{align*} So by sandwich rule, \(\nu _p(m!)\to \infty \).

Question 2. Does the sum

\[ \sum _{m=0}^\infty m \]

converge \(p\)-adically?   

The series converges only if \(\nu _p(m)\to \infty \). But taking for some \(n\in \N \), \(\nu _p(dp+1)=\nu _p(1)=0\not \to \infty \). Thus the series does not converge \(p\)-adically.

Question 3. Does

\[ \sum _{m=0}^\infty \frac {(3m)!}{(2m)!} \]

converge \(p\)-adically.   

\[ \frac {(3m)!}{(2m)!}=m!\cdot \binom {2m}{3m} \]

So the series converges \(p\)-adically.

Question 4. Does

\[ \sum _{m=0}^\infty \frac {p^{m^2}}{(m!)!} \]

converge \(p\)-adically.   

This question is equivalent to asking if \(\nu _p((m!)!)\) is less or greater than \(m^2\). But using the previous calculation,

\[ \nu _p((m!)!)\le \frac {m!}{p-1} \]

The lecturer treats this as is if it were an equality, so concludes the series does not converge \(p\)-adically.

Question 4. Does

\[ \exp (p)=\sum _{m=0}^\infty \frac {p^{m}}{m!} \]

converge \(p\)-adically.   

\[ \nu _p\frac {p^m}{m!}=m-\frac {m}{p-1}\to \infty \]

if and only if \(p\ne 2\).

  • Remark. Where the lecturer has been treating inequalities as equalities, he really meant

    \[ (-1)+\frac {m}{p-1}\le \nu _m(m!)\le \frac {m}{p-1} \]

Question 4. Does

\[ \exp (\sqrt {p})=\sum _{m=0}^\infty \frac {\sqrt {p}^{m}}{m!} \]

converge \(p\)-adically.   

Here \(\nu _p(\sqrt {p})=\frac 12\). Then

\[ \nu _p(a_n)=\frac {m}2-\frac {m}{p-1} \]

so the series converges for \(p\ne 2,3\).

Firstly, let us reconsider an example from last week, the \(p\)-adic valuation

\[ \nu _p(m!)\approx \frac {m}{p-1} \]

To be precice, we should use the inequality

\[ \frac {m}{p-1}-\left (\frac {\log (m)}{\log (p)}+1\right )\le V_p(m!)\le \frac {m}{p-1} \]

In fact, if \(m=a_0+a_1p+\cdots +a_kp^t\) for \(0\le a_i\le p-1\), then

\[ \nu _p(m!)=\frac {m-s}{p-1}\quad \text {where} \quad s=a_0+\cdots +a_k \]

This gives the upper end of the inequality. Note that \(k=\floor {\frac {\log m}{\log p}}\) and \(0\le s\le k(p-1)\)

\[ \nu _p(m!)=\sum _{i\ge 1}\floor {\frac {m}{p^i}} \]

\[ \frac {m}{p-1}=\sum _{i\ge 1} \frac {m}{p^i} \]

\(\seteqnumber{0}{0.}{0}\)

  • Example 2. Show

    \[ \sum _{m\ge 0}\binom {2m}{m} \]

    does not converge \(p\)-adically for any prime \(p\). (This is the last homework question.)

    For two integers \(a\ge b\),

    \[ \binom {b}{a}=\frac {a!}{b!(b-a)!} \]

    so

    \[ \binom {2m}{m}=\frac {(2m)!}{m!m!} \]

    we need to show this does not go to \(\infty \) as \(m\to \infty \). To do this, we will show that for \(m=p^k\),

    \[ \nu _p\binom {2m}{m}=0\text { or }1 \]

    Here \(m=0\cdot 1+0\cdot p+\cdots +1\cdot p^k\), so

    \[ \nu _p(m!)=\frac {m-s}{p-1}=\frac {p^k-1}{p-1} \]

    and

    \[ \nu _p(2m!)=\frac {2m-s}{p-1}=\frac {2p^k-2}{p-1} \]

    unless \(p\) is odd, in which case \(s=1\).

    Therefore,

    \[ \nu _p\binom {2m}{m}=\nu _p\left (\frac {(2m)!}{m!m!}\right )=\nu _p(2m!)-2\nu _p(m!)=\frac {2p^k-(1\text { or }2)}{p-1}-2\frac {p^k-1}{p-1}=0\text { or }1 \]

  • Example 3. For \(x\in \R \), define

    \[ \binom {x}{m}:=\frac {x(x-1)\cdots \big (x-(m-1)\big )}{m!} \]

    Recall that

    \[ (1+x)^\frac 1p=\sum _{m\ge 0}\binom {\frac 1p}{m}x^m \]

    Use this to prove that \((1+p^2x)^\frac 1p\) converges \(p\)-adically when \(x\in \Z \) or even \(x\in \Z _{(p)}\)

    Clearly we want to show that

    \[ \nu _{p}\left (\binom {\frac 1p}{m}(p^2x)^m\right )\to \infty \quad \text {as}\quad m\to \infty \]

    Let this summand be \(a_m\).

    \(\seteqnumber{0}{0.}{0}\)

    \begin{align*} a_m&=\binom {\frac 1p}{m}p^{2m}x^m\\ &=\frac {(1/p)(1/p-1)\cdots (1/p-(m-1))}{m!}p^{2m}x^m\\ &=\frac {1(1-p)\cdots (1-p(m-1))}{m!}p^mx^m \end{align*} \(p\) does not divide the numerator, so

    \[ \nu _p(a_n)=+\nu (p^m+\nu )p(x^m)\ge m-\frac {m}{p-1}=m\frac {p-2}{p-1}\to \infty \quad \text {as}\quad p\to \infty \]

    for \(p\ne 2\).

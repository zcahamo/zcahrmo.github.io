Question 2. Compute \(\displaystyle \left (\frac {37603}{48611}\right )\). 48611 is prime, and

\[ 37603=31\times 1213 \]

  

We know

\[ \left (\frac {37603}{48611}\right )=\left (\frac {31}{48611}\right )\left (\frac {1213}{48611}\right ) \]

But prime factorisations can be hard to compute if a number has few prime factors, wo we need a better method.

Generalised quadratic reciprocity:

  • Definition (Jacobi Symbol). If \(b\) is an odd positive integer with prime factorisation \(b=p_1\cdot p_2\cdots p_n\), and \(a\) is an integer, the Jacobi Symbol is

    \[ \left (\frac {a}b\right )=\begin {cases} \left (\frac {a}{p_1}\right )\cdots \left (\frac {a}{p_n}\right )&\gcd (a,b)=1\\ 0&\text {otherwise} \end {cases} \]

    This takes the values \(0,\pm 1\).

This is a generalisation of the Legendre Symbol. It has the properties

\(\seteqnumber{0}{0.}{0}\)

\begin{align*} \left (\frac {aa'}b\right )&=\left (\frac {a}b\right )\left (\frac {a'}b\right )\\ \left (\frac {a}{bb'}\right )&=\left (\frac {a}b\right )\left (\frac {a}{b'}\right ) \end{align*} If \(a,b\) are odd integers, then

\(\seteqnumber{0}{0.}{0}\)

\begin{align*} \left (\frac {37603}{48611}\right )&=(-1)\left (\frac {48611}{37603}\right )\\ &=(-1)\left (\frac {11008}{37603}\right )\\ &=(-1)\left (\frac {2}{37603}\right )^8\left (\frac {43}{37603}\right )\\ &=(-1)(+1)(-1)\left (\frac {37603}{43}\right )\\ &=(-1)(+1)(-1)\left (\frac {21}{43}\right )\\ &=(-1)(+1)(-1)(+1)\left (\frac {43}{21}\right )\\ &=(-1)(+1)(-1)(+1)\left (\frac {1}{21}\right )\\ &=(-1)(+1)(-1)(+1)(+1)=1 \end{align*} Alternatively, spotting that \(21+43=64=8^2\) with save a few lines.

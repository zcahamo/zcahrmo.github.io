\(R\) is a commutative ring. An ideal is \(I\subseteq R\) such that

\(\seteqnumber{0}{0.}{0}\)

\begin{gather} I\normal (R,+,0)\\ \forall r\in R,\ \forall i\in I\quad ri\in I \end{gather}

  • Example 1. For \(R=\Z \), \(\{0\},\Z \) are ideals, and so too are

    \[ (n)=\{an:a\in \Z \}\quad n\in \N \]

An ideal \(I\) is principal if and only if

\[ \exists r\in R\quad I=(r) \]

An integral domain in which all ideals are principal is a P.I.D.

If \(I\) is an ideal in \(R\), then the quotient ring

\[ R/I=\{a+I:a\in R\} \]

is a ring.

\(\seteqnumber{0}{0.}{2}\)

\begin{gather*} (a+I)+(b+I)=(a+b)+I\\ (a+I)\cdot (b+I)=ab+I\\ O_{R/I}=I=i+I\quad i\in I\\ 1_{R/I}=1+I \end{gather*}

  • Example 2. Consider \(\Z /(n)\) for \(n\in \N \), or

    \[ \F [x]/(p(x)) \]

    the ring of polynomials over \(\F \) modulo \(p(x)\)

Let \((a),(b)\) be ideals in \(\Z \), \(a,b\in \Z \). Then

\[ (a)+(b)=\{ha+kb:h,k\in Z\}=:(a,b) \]

Since \(\Z \) is a P.I.D. we may write this as a principle ideal \((d)\) for \(d=\gcd (a,b)\). \(\Z \) is a principle ideal domain because of the division algorithm, and for the same reasons, so are polynomial rings.

  • Definition. An ideal \(I\ne R\) is maximal in \(R\) if and only if whenever \(i\subseteq J\subseteq R\), an ideal for an ideal \(J\), then \(J=R\) or \(J=I\).

  • Theorem 0.1. If \(I\) is an ideal in \(R\) then \(R/I\) is a field if and only if \(I\) is maximal.

    \[ (R/I)^\times =R/I-\{0\} \]

One of the main applications of ring theory is the construction of fields, and for that we consider ring homomorphisms. A ring homomorphism \(\phi :R\to S\) satisfies

\(\seteqnumber{0}{0.}{2}\)

\begin{align} \phi (a+b)&=\phi (a)+\phi (b)\\ \phi (ab)&=\phi (a)\phi (b)\\ \phi (1_R)&=1_S \end{align}

  • Example 3. If \(\phi :\Z \to R\) for a ring \(R\) is a ring homomorphism, then

    \(\seteqnumber{0}{0.}{5}\)

    \begin{gather*} \phi (0)=0\\ \phi (1)=1_R\\ \phi (n)={1_R+\cdots +1_R}\quad \text {($n$ times)}\quad n\ge 1\\ \phi (n)={-1_R+\cdots -1_R}\quad \text {($n$ times)}\quad n\le -1 \end{gather*}

    Hence we see that the properties of a ring homomorphism are actually quite restrictive.

  • Theorem 0.2 (1st Isomorphism Theorem). If \(\phi :R\to S\) is a ring homomorphism, then

    \[ R/\ker (\phi )\cong \im (S) \]

    since \(\ker (\phi )\) is an ideal

    \[ \ker (\phi )=\{r\in R:\phi (r)=0\} \]

Consider \(\Q [\alpha ]\) for \(\alpha \in \C \), where this is the smallest ring containing \(\Q \) and \(\alpha \). Then

\[ \Q [\alpha ]=\{p(\alpha ):p(x)\in \Q [x]\} \]

Suppose

\[ \exists p(x)\in \Q [x]\quad p(\alpha )=0 \]

Let \(m(x)\in \Q [x]\) be of minimum degree to satisfy \(m(\alpha )=0\). Then the lecturer claims

\[ \Q [\alpha ]\cong \Q [x]/m(x) \]

  • Proof. Take the map

    \(\seteqnumber{0}{0.}{5}\)

    \begin{gather*} \phi :\Q [x]\to \Q [\alpha ]\\ \phi (p(x))=p(\alpha ) \end{gather*} This is a surjective ring homomorphism. Then let us prove

    \[ \ker (\phi )=(m(x)) \]

    We know this kernel \(K\) is ideal. Let \(p(x)\in K\). If \(p(x)=0\) then \(p(x)\in (m(x))\). Otherwise, use polynomial division

    \[ p(x)=g(x)m(x)+r(x)\qquad \deg r<\deg m \]

    Then

    \[ 0=p(\alpha )=q(\alpha )0+r(\alpha )=r(\alpha ) \]

    so the minimality of \(m\) implies \(r(x)=0\). Therefore

    \[ p(x)=q(x)m(x)\in (m(x)) \]

    so \(K\subseteq (m(x))\). Trivially

    \[ m(\alpha )=0\implies (m(x))\subseteq K \]

    so \(K=(m(x))\). Thus the result follows by the 1st Isomorphism Theorem. □

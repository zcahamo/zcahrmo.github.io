  • − 1st definition of compactness - finite subcover.

  • − Compact means bounded and closed. inverse true for \(\R ^n\).

  • − Compactness is equivalent to sequential compactness - any sequence in \(S\) has a convergent subsequence in \(S\). (not true in topological spaces)

W4 stuff: \(D\) is dense in \(X\) if every \(x\in \backslash D\) is a limit point of \(D\)

\[ \forall r>0\ \exists y\in D\quad y\in B^o(x,r) \]

Intuatively, wherever you look, there is a point.

In terms

\[ \forall y_n\in D\quad y_n\to x\ \ \text {as}\ \ n\to \infty ,\quad \text {so}\ d(y_n,x)\to 0 \]

\(X\) is always dense, so is a dense subset of every set.

\(\ell ^\infty \) is a very big space. It is not separable. Inside \(B^o(0,2)\), there are uncountably many disjoint open balls of radius \(\frac 12\). Compare this to \(\R ^2\), where you can only fit finitely many.

What is a base? Useful concept because it generalises to topological spaces. However, in metric spaces, the only bases we consider are collections of balls. Bases are like building blocks of the space. Examples of bases (not different from basis) in \(\R ^2\) are

  • − The set of all open balls

In the proof, we don’t take all the balls in \(D\), but we do take just enough.

Difference between a compact space and a compact subset of the space, the two are trivially related.

What’s the point of all this stuff? To prove the theorem: a set is compact iff it is closed and totally bounded.

What’s the relationship between bounded and totally bounded? Totally bounded if has finite \(\varepsilon \)-net, for any \(\varepsilon >0\) (union of finite number of balls, radius \(\varepsilon \)).

\[ S\text { is bounded }\iff \exists \varepsilon >0\quad S\text { has a finite $\varepsilon $-net} \]

(in fact, iff it can be covered in a single ball) So the difference is \(\exists \) and \(\forall \).

Think back to \(\ell ^\infty \). This is a long way from being totally bounded. This doesn’t have a countable \(\frac 12\)-net, let alone a finite one.

  • Example. Question about the \(\ell ^\infty \) example, slide 2 of W4. Given \(G\in 2^\N \) (the set of all subsets of \(\N \). This notation is used since \(2^X\) has size \(2^{\abs {X}}\). We could have just written \(G\subset \N \).) Let \(e^G\in \ell ^\infty \). Take the sequence.

    \[ e^{\{2,3,4\}}=(0,1,1,1,0,0,0,0,0,\dots ) \]

    This notation has nothing to do with exponential. It just means the sequence has 0s in places \(2,3\) and \(4\). Also take the set

    \[ e^{\{1,4\}}=(1,0,0,1,0,0,0,0,0,\dots ) \]

    Then

    \[ (e^G-e^{G'})_n=\begin {cases} 0&\text {if}\ n\in G\cap G'\\ \pm 1& \text {otherwise} \end {cases} \]

    For our particular case,

    \[ (e^{\{2,3,4\}}-e^{\{1,4\}})_n=(-1,1,1,0,0,0,\dots ) \]

    So

    \[ \norm {e^G-e^{G'}}_\infty =\sup _{n\in \N }\abs {(e^G-e^{G'})_n}=\begin {cases} 1&\text {if}\ G\ne G'\\ 0& \text {otherwise} \end {cases} \]

  • Example. (Sheet 4 Q5) Show that every separable metric space \((X,d)\) embeds isometrically into \(\ell ^\infty \), that is

    \[ \exists I:X\to \ell ^\infty \ \ \forall x,y\in X\qquad \norm {I(x)-I(y)}_\infty =d(x,y) \]

    so any separable metric space fits in \(\ell ^\infty \), which is non-separable, whilst preserving the metric structure.

    Let \(\{x_1,x_2,x_3,\dots \}\) be a countable dense subset of \(X\), and let \(x_*\in X\) be an arbitrarily chosen “reference point”. Define for all \(x\in X\),

    \[ I(x)=\Big ( d(x,x_1)-d(x_1,x_*),d(x,x_2)-d(x_2,x_*),d(x,x_3)-d(x_3,x_*),\dots \big ) \]

    In any metric space, we have the reverse \(\triangle -\)ineq.

    \[ \forall x,y,z\in X\quad \abs {d(x,y)-d(x,z)}\le d(y,z) \]

    This may be proven using \(\triangle -\)ineq and symmetry of \(d\). So

    \[ \forall n\in \N \quad \abs {d(x,x_n)-d(x_n,x_*)}\le d(x,x_*)<\infty \]

    Hence \(I(x)\in \ell ^\infty \).

    Now use reverse \(\triangle -\)ineq to prove

    \[ \forall x,y\in X\quad \norm {I(x)-I(y)}_\infty \le d(x,y) \]

    And then apply density of \(\{x_1,x_2,\dots \}\) in \(X\) to prove

    \[ \norm {I(x)-I(y)}_\infty \ge d(x,y) \]

Clarification. In a norm space \((V,\norm {\cdot })\), unless stated otherwise, consider the induced metric \(d(x,y)=\norm {x-y}\). This may not always be the case; a different metric such as the Railway metric may be specified, but otherwise assume the default.

Similarly, for an inner product space \((V,\langle \cdot ,\cdot \rangle )\), the default norm is \(\norm {V}=\sqrt {\langle V,V\rangle }\), and so the default metric is

\[ d(x,y)=\norm {x-y}=\sqrt {\langle x-y,x-y\rangle } \]

  • Example 0.1. Why do sets in a discrete space have no limit points? Remember that if \(r<1\), then

    \[ B^o(x,r)=\{x\} \]

    so no other elements \(y\ne x\) are in \(B^o(x,r)\).

  • Proof. To show if a set is closed, then it contains all its limit points. Consider a point \(x\) not contained in \(A\). For closed \(A\), \(X\backslash A\) is open. So

    \[ B^o(x,r)\cap A=\emptyset \]

    So \(x\) is not a limit point. □

The words “not complete” is quite descriptive, there is something missing. \((0,1)\) with \(d(x,y)=\abs {x-y}\) is not complete since it is missing the endpoints.

(Not examinable) The completion of \(\big (C[a,b],\ \norm {\cdot }\big )\) is called \(L^1(a,b)\). See Measure Theory, Variational Methods, Linear PDEs for more on this.

  • Example 0.2. Reconsider the railway metric with the Cauchy property. For \((V,\norm {\cdot })\) with

    \[ d(x,y)=\begin {cases} \norm {x}+\norm {y}&x\ne y\\ 0&x=y \end {cases} \]

    • iii) Prove \((V,d)\) is complete.

      If \(x_n\ne x_m\), then \(d(x_n,x_m)=\norm {x_n}+\norm {x_m}\), and if \(x_n=x_m\), then \(d(x_n,x_m)=0\). So for Cauchy sequence \(\langle x_n\rangle \), there are two cases.

      • a) \(\norm {x_n}\to 0\) as \(n\to \infty \). In this case, \(x_n\to 0\) (see the previous class.)

      • b) \(x_n\) is eventually constant, say \(x_n=x\) for \(n<N\). Then clearly \(x_n\to x\).

      Both possibilities give a limit in \(V\), so \((V,d)\) is complete.

Pointwise vs Uniform convergence Pointwise may be characterised as follows

\[ f_n\to f\text { pointwise}\iff \text {for each fixed }x,\quad f_n(x)\to f(x)\ \text { as }n\to \infty \]

Pointwise convergence is not about sequences of points, its about different cases with different, fixed, \(x\).

  • Example 1. For \(X=[0,1]\), show \(f_n(x)=x^n\) converges pointwise to

    \[ f(x)=\begin {cases} 1&x=1\\ 0&x\in [0,1) \end {cases} \]

    • − Case 1 is when \(x=1\). Here \(f_n(x)=1\) for all \(n\), so \(f_n(x)\to 1\) as \(n\to \infty \).

    • − Case 2 is when \(x\in [0,1)\). Here

      \[ f_n(x)=x^n\to 0\quad \text {as}\quad n\to \infty \]

Uniform convergence on the other hand may be characterised as

\[ f_n\to f\text { uniformly}\iff \norm {f_n-f}_{\sup {}}=\sup _{x\in X}\abs {f_n(x)-f(x)}\to 0\ \ \text {as}\ \ n\to \infty \]

The whole of \(X\) is being considered here in one go.

  • Example 2. Consider \(X=[0,1]\) and \(f_n\) continuous on \([0,1]\), increasing linearly from \(f_n(0)=0\) to \(f_n\big (1/(2n)\big )=n\), and then decreasing linearly to \(f_n(1/n)=0\).

    First let us show this converges pointwise on \([0,1]\) to 0.

    • − For \(x=0\), \(f_n(x)=0\) for all \(n\), so \(f_n(x)\to f(0)=0\) as \(n\to \infty \).

    • − For \(x\in (0,1]\), for \(n>\frac 1x\), we have \(f_n(x)=0\), so \(f_n(x)\to 0\) as \(n\to \infty \). Notice that it does not actually matter what \(f_n(x)\) is in \((0,1/n)\), this is a red herring.

    Is this convergence uniform? No, as can be shown by a counterexample. This is some sequence \((x_n)\in X\) such that

    \[ \abs {f_n(x_n)-f(x)}\not \to 0 \]

    Take the sequence \(x_n=\frac 1{2n}\). Then

    \[ \abs {f(x_n)-f(x)}=\abs {n-0}=n\to \infty \]

  • Example 3. Taking the previous example but with \(f\big (1/(2n)\big )\) to \(\frac 1n\) changes this. It still converges pointwise, by the same argument, but now

    \[ \norm {f_n-f}_{\sup {}}=\norm {f_n}_{\sup {}}=\frac 1n\to 0\quad \text {as} n\to \infty \]

    so this function converges uniformly too.

  • Remark. If you’re asked to evaluate

    \[ \lim _{n\to \infty } \int _a^b f_n(x)\;\d x \]

    then by the theorem in the notes, this is

    \[ \int _a^b f(x)\;\d x \]

    provided that \(f_n\to f\) uniformly on \([a,b]\) and \(f_n\) is Riemann integrable for each \(n\).

  • Remark.

    \[ \norm {g_n}_{\sup {}}\nimplies \sum _{n=1}^\infty g_n\text { converges} \]

    A counterexample is given by

    \[ g_n(x)=\frac 1n\quad \text {for}\quad x\in [0,1] \]

    since

    \[ \norm {g_n}_{\sup {}}\frac 1n\to 0\quad \text {but}\quad \sum _{n=1}^\infty g_n(x)=\sum _{n=1}^\infty \frac 1\infty =\infty \]

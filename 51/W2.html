Question 1. (2020 exam Q1) Let \((U,\norm {\cdot })\) be a normed space and define

\[ d(x,y)=\begin {cases} \norm {x}+\norm {y}& x\ne y\\ 0&x=y \end {cases} \]

This is called the railway metric; to get from \(x\) to \(y\), one has to travel via 0.

  • a) Prove that \((U,d)\) is a metric space.

  • b) Describe all convergent sequences in this space.

  • c) Describe \(B^o(0,1),B^o(1,1),B^o(2,3)\) in the space when \(V=\R \) and \(\norm {\cdot }=\abs {\cdot }\)

  2 marks

  • a) Must satisfy each part of the definition. Trivial cases, particularly \(x=y\) have been omitted for brevity.

    • i) \(\norm {x}\ge 0\), so \(x,y\) \(d(x,y)\ge 0\). Also,

      \[ x\ne y\implies d(x,y)\ge \min \big (\norm {x},\norm {y}\big )>0 \]

      so \(d(x,y)=0\implies x=y\)

    • ii) For \(x\ne y\),

      \[ d(x,y)=\norm {x}+\norm {y}=\norm {y}+\norm {x}=d(y,x) \]

    • iii) For \(x,y,z\) all distinct,

      \[ d(x,y)=\norm {x}+\norm {y}\le \norm {x}=\norm {Z}+\norm {z}+\norm {y}=d(x,z)+d(z,y) \]

  • b) Consider the following cases:

    • i) If the limit \(x\) is the origin \((x=0)\), then

      \[ d(x_n,0)=\max \big (0,\norm {x_n}\big ) \]

      so

      \[ x_n\to 0\text { in }(U,d)\iff \norm {x_n}\to 0 \]

    • ii) If the limit \(x\ne 0\), then for \(x_n\ne x\),

      \[ d(x_n,x)=\norm {x_n}+\norm {x}\ge \norm {x}>0 \]

      So by the definition of convergence,

      \[ x_n\to x\iff x_n=x\text { eventually} \]
